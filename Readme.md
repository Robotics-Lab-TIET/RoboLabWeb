# How to use 
**fork  and clone this repo,**
do
```bash
cd robolabweb
```

install dependices
```bash
npm i 
```

create .env file,<br>
copy past template from .env.sample and fill in ur data
<br>to start server do
 ```bash 
 npm start // for production
 npm run dev // for development 
 ```
