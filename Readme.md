# How to use 
**fork  and clone this repo,**
do
```bash
cd robolabweb
```

install dependices
```bash
npm i 
```

create .env file
specify port as PORT and MongoDb srv as 
DB_URI=put Your srv here 
 **to start server do**
 ```bash 
 npm start
 ```

by default npm start will start nodemon on app.js


For more information,  [please read this ](plane.md).
